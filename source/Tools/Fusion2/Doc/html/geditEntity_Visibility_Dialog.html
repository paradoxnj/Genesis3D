Entity Visibility Dialog

The Entity Visibility Dialog (Entities|Visibility…) allows you to select which types of entities are displayed by the editor in the orthographic and textured views. To hide all entitys of a particular type, clear the checkbox next to the entity’s type name in the Visibility dialog box. To enable display of the entity type, check the box next to the entity’s type name.

When disabled for an entity type, this setting prevents all entities of that type from being displayed. When this setting is enabled for an entity type, entities of that type are displayed subect to the group visibility rules for the group to which the entities belong.
