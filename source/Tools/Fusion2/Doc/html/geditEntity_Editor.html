Entity Editor

The Entity Editor allows you to select an entity and change its properties. To select an entity for editing, pick it from the Entity Editor dialog box’s drop-down combo box. Once selected, the entity’s properties and associated values are displayed below the drop-down combo box. To edit a property, double-click on it, or highlight it by moving the selection bar and then press Enter.

Entity properties can be of many different types: number, string, position, color, or model. When you double-click on a property, the program will display that property in a dialog box that is specific to that property’s type. For example, if you double-click on a color property, the color picker dialog is displayed.
