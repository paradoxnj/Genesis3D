Template

The Template Options Page allows you to select items to be placed in the level. You can select from one of six different geometry primitives, any of the defined entity types, or any premade objects that are stored in the program’s Objects directory.

The available geometry primitives are Box, Spheroid, Cylinder, Staircase, Arch, and Cone. To select one, put the editor into Template mode, and select one of the primitives’ buttons on the Template Options Page. Once you’ve selected and configured the primitive, you can shape its template (which is displayed in blue in the orthographic views) and then press Enter to place it into the level.

To place an entity in the level, select the desired entity from the entities drop-down combo box and click on the entity button (the lightbulb icon next to the drop-down). The template entity is displayed in blue in the orthographic views. You can move the template entity in the orthographic views and press Enter to place it in the level.

You can also place an entity by clicking in the 3D view where you want the entity placed. The entity will be placed 16 texels from the surface on which you clicked.

To place a premade object, select the object from the objects drop-down combo box and click on the premade object button next to the drop-down. The object’s center position will show as a blue “X” in the orthographic views. You can move it as though you were moving an entity. When you press Enter to place the object in the level, it will be centered on the “X”.
