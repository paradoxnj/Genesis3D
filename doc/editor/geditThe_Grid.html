The Grid

In the orthographic views, the grid is displayed in order to give you some sense of scale, and also to help you align your geometry. The status line at the bottom of the editor window displays the current fine grid size and the current grid snap size. The size of the fine grid will change as you zoom in and out in the orthographic views.

The Grid dialog allows you to specify the grid boundaries on which geometry should snap when it is moved or resized.
