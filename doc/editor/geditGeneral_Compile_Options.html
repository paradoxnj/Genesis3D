General Compile Options

Entities Only
Perform a quick compile. See Quick Compile for more information.
Suppress Hidden Brushes
If checked, brushes and entities that are in hidden vis groups will not be written to the .MAP file, and as a result will not be included in the compiled level.
Vis Detail Brushes
When checked, brushes marked with the Detail attribute will be taken into account during the Vis stage. This option is normally off to speed level compilation. You should turn it on for the final compile.
BSP
When checked, creates level geometry from data exported by the editor.
Vis
When checked, performs potential visible set calculations on compiled level.
Light
When checked, performs calculations to add lighting effects to the level.
Preview
If checked, the program will automatically prompt you to run the level previewer after a successful compilation.
